### TypeScript 数据类型

1. 所有JavaScript类型

2. 六个新类型：
- any
- unknow
- never
- void
- tuple
```ts
let person: [string, number] = ["Alice", 25];
```

- enum 枚举：用来定义一组命名常量
```ts
enum Color {
  Red = 'rgba(0,0,0)',
  Green = 'rgba(0,0,0)'
  Blue = 'rgba(0,0,0)'
}

enum Animal {
  Dog,
  Cat
}

let favoriteColor: Color = Color.Green

```
