### HTTP
超文本传输协议，明文传输，实现网络通信的规范协议

### HTTPS
HTTP + SSL，通过SSL证书来验证服务器的身份，为浏览器和服务器之间的通信加密

传输流程：使用“证书”安全的交换一个临时的“会话密钥”，然后用这个密钥来加密所有后续的通信。

如何加密：
- 对称加密：加密和解密使用同一个密钥
- 非对称加密：两个密钥，一个公钥，一个私钥。公钥公开，私钥保密。互相加密解密
- 摘要算法：保证信息完整可信，未被篡改
- 数字签名：保证消息确实是由发送方签名并发出来的

### 区别：
- 安全性：HTTPS比HTTP更安全，HTTP是明文传输，HTTPS是加密传输
- 默认端口： HTTP默认80，HTTPS默认443
- 费用：HTTPS需要SSL证书，费用高
